A diverse set of departments

Web development

Students learn about web design principles, HTML, CSS, JavaScript, and other programming languages used for web development, as well as server-side programming, database integration, and web security.

Android development

Students learn about the Android architecture, user interface (UI) design, navigation, and layout, as well as the fundamentals of programming languages like Java or Kotlin, used for Android development. They may also learn about integrating data from various sources, such as databases, web services, or APIs, and how to implement security features like user authentication, authorization, and data encryption.

Data Science, Machine Learning and Artificial Intelligence

Students learn about data preprocessing and cleaning, exploratory data analysis, statistical inference, and predictive modeling using various algorithms and techniques. They may also learn about programming languages and tools used for data science, such as Python, R, SQL, and Jupyter Notebook. Additionally, the course may cover topics related to machine learning, such as supervised and unsupervised learning, neural networks, decision trees, and clustering. The course may also cover Artificial Intelligence topics such as Natural Language Processing, Computer Vision, and Reinforcement Learning.

Cyber-Security

Students learn about common cyber threats, such as malware, phishing, and denial-of-service attacks, and how to prevent them through network security, cryptography, and access control measures. They may also learn about ethical hacking and penetration testing techniques, to identify and exploit vulnerabilities in computer systems and networks before malicious hackers do. Additionally, students may learn about compliance and regulations related to cyber-security, such as GDPR, HIPAA, and PCI DSS.

User Interface/User Experience (UI/UX)

Students learn about the principles of human-computer interaction (HCI), user research and testing, prototyping, and visual design. They may also learn about various design tools and technologies used in UI/UX, such as Adobe XD, Sketch, Figma, and InVision. Additionally, students may learn about the psychology of user behavior, usability testing, and accessibility, to ensure that digital products are usable and accessible to a wide range of users.
